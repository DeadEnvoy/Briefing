# Briefing

## Description

This mod adds a briefing screen that shows player info, current date, location, and days since the Knox Event when you load your save.

![Briefing demonstration](https://i.postimg.cc/6qq0W7Q3/ezgif-8e4ca4293d2827-1.gif)

Days since outbreak are only tracked when using proper lore-friendly dates — otherwise it'll just show how many days you've survived.

When loading into the world, the game pauses during the briefing animation, giving the game extra time to load everything properly and preventing those annoying micro-stutters when you start playing.

This mod **WON'T** work in tutorial or challenges.
